Synopsis¶

Get specified number of free addresses; Provision to reach requested number.

                Examples¶

                # Add public IP block
- dimensiondata_public_ip_block:
    region: na
    location: NA5
    network_domain: test_network
    action: add
# Delete public IP Block by base IP.
- dimensiondata_public_ip_block:
    region: na
    location: NA5
    network_domain: test_network
    action: delete
    base_ip: 168.128.2.100
# Delete public IP Block by block ID.
- dimensiondata_public_ip_block:
    region: na
    location: NA5
    network_domain: test_network
    action: delete
    block_id: 6288ab1c-0000-0000-0000-b8ca3a5d9ef8
